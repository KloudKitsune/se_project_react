# WTWR (What To Wear)

WTWR is a React frontend project that recommends clothing based on the current weather. It fetches weather data from the OpenWeather API and displays clothing items filtered for the current conditions.

## Features

- Fetches current weather for a specified location
- Displays current location and temperature
- Filters clothing items based on weather (hot, warm, cold)
- Opens modals to preview items or add new garments
- Responsive and styled using CSS

## Tech Stack

- React + Vite
- JavaScript (ES6+)
- CSS
- OpenWeather API

## Getting Started

1. Clone the repo: using bash do the following

   1. git clone git@github.com:KloudKitsune/se_project_react.git
   2. cd se_project_react
   3. npm install
   4. npm run dev
